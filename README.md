Статический сайт с 3D-кубиком Рубика: случайный расклад, ручные повороты,
сброс, автосборка, смена палитры и победная анимация.

## Запуск

```bash
python3 -m http.server 5173
```

После запуска открой:

```text
http://127.0.0.1:5173/
```

Все зависимости лежат локально в папке `vendor`, поэтому сборка не нужна.
